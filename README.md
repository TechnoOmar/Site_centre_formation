# Application de Concurrences Sportives

## 📌 Description
Application desktop de gestion des compétitions sportives développée avec Delphi XE7.  
Elle permet de gérer les équipes, les matchs, les résultats et les classements.

---

## ⚙️ Fonctionnalités
- Gestion des équipes et joueurs
- Organisation des compétitions
- Gestion des matchs
- Saisie des résultats
- Classement automatique

---

## 🛠️ Technologies utilisées
- Embarcadero Delphi XE7
- Windows Desktop Application

---

## 💻 Systèmes supportés
- Windows 10
- Windows 11
- Windows Server
